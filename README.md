# radio_online
Radio Ciocile: Aplicație web completă pentru streaming radio, construită cu PHP, MySQL și JS. Oferă sincronizare audio perfectă între toți ascultătorii, player hibrid (MP3 + YouTube), chat live și sistem de reacții animate în timp real. Include un CMS robust pentru managementul playlist-ului și statistici. Design modern, glassmorphism și 100%
